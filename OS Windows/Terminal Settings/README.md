# Terminal Themes (Windows 'Terminal')
https://github.com/microsoft/terminal

## Step 1: Open Windows Terminal settings JSON

1. Open Windows Terminal
2. Press `Ctrl + Shift + ,` OR
   1. Click Drop Down Icon Next to "+" -> 'Settings'
   2. Click “Open JSON file” (Bottom Left on Window)

## Step 2: Add the Color Scheme & Defaults

1. Find (or create) the "profiles" array and pase entire "defaults" object inside it
2. Find (or create) the "schemes" array and paste entire "schemes" object inside it
